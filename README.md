## Quick Start

```bash
# 1. Copy env file and fill in agent URLs
cp .env.example .env

# 2. Build the orchestrator image
docker build -t yard-triage:latest .

# 3. Run
docker run --env-file .env -p 9000:9000 yard-triage:latest

# 4. Invoke
curl -X POST http://localhost:9000/invoke \
  -H "Content-Type: application/json" \
  -d '{"input": {"message": "hello"}}'

# 5. Health check
curl http://localhost:9000/health
```

## Editing

This code is fully editable. The orchestrator calls agents over HTTP (A2A protocol) — you can modify the orchestration logic without touching the agents.

To re-import into AgentYard Studio:
```bash
curl -X POST http://localhost:8080/api/systems/import \
  -H "Content-Type: application/x-yaml" \
  --data-binary @system.yaml
```
